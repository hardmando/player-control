# Project
Player Control

# Goal
Create macOS-like control centre player to control playback across whole system (Arch-centered)

# Stack
Rust + Bash

# Features
- play/pause (if provided by application)
- volume next/previous (if provided by application)
- display all soures of playback, i.e. all applications that are playing media, if browser - treat each tab that is playing media as separate app

# Constraints
- must be able to launch from waybar
- must work primary on Arch Linux based distros

# Style
TUI visual style, Omarchy volume controls like
